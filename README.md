# Projeto WorldTech

Automação de precificação de produtos com integração WhatsApp e WordPress.

## O que esse projeto faz

- Lê listas de preços de fornecedores recebidas via WhatsApp
- Compara preços entre fornecedores e seleciona o melhor
- Aplica margem de lucro automaticamente
- Envia lista atualizada para clientes via WhatsApp
- Atualiza preços dos produtos no site WordPress/WooCommerce

## Tecnologias

- Python
- Evolution API / Z-API (WhatsApp)
- WooCommerce REST API (WordPress)
